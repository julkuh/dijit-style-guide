# dijit-style-guide
A nicer way to present dojo's default Theme-Tester

## Getting Started

1. Install globally with npm `bower` and `grunt-cli`.
2. Run `bower install` to install Dojo and Dijit dependencies.
3. Run `grunt` to open the Dijit theme tester.
